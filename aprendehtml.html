Afinal de contas o que é HTML se já falamos que ele não é uma linguagem de programação? A sigla HTML significa
Hypertext Markup language.Que em português significa linguagem de marcação de texto.HTML nada mais é do que todos
os textos que estão escritos na web esse aqui também é um exemplo vivo desse tipo de texto.Podemos utilizar também
o XML ao qual tem a mesma funcionalidade.

  2. Aprenda o básico de git e github para já mostrar o seu aprendizado para as empresas

    Criando repositório local e adiciona-lo ao nosso Git-hub remoto

    1-Passo:Crie uma pastinha com o nome do projeto .
    Exemplo: ->meu-repositorio-teste
    2-Passo: Abrir o VS e criar o projeto de preferência.

    3-Passo:File ->open folder ->Salvar na pasta criada anteriormente

    4-Passo:No VS crie uma pasta REDME.md .(.md é arquivo markdown)

    5-Passo: digite na pasta
    # Nome da pasta
    ## Descrição

    Detalhes

    6-Passo:Entrar na conta do Git Hub e criar um repositório com o nome
    identico ao qual criamos a pasta local .
    Observação :Não adicione o README.md

    7-Passo:A tela aparecera os próximos passos mas vamos lista-lós aqui.

    8-Passo:Entre na pasta local e abra o git bash dentro dela.

    9-Passo:Digite para iniciar o repositório lá dentro .
    $git init

    10-Passo:digite
    $git add README.md

    11-Passo :digite o commit para especificar o projeto.
    $git commit -m "modificação"

    12-Passo:digite;
    $git branch -M main

    13-Passo:copie o link do repositório gerado na página do git hub após.
    a realização da criação do repositório.

    14-Passo:escreva no terminal do Git bash
    $git remote add origin link

    16:Passo: Vamos empurrar todo o código para o github.
    $git push -u origin main

    19-Passo:Atualize a página do git-hub
